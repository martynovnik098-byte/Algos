**Задача 1639. Шоколад 2**

*   **Суть:** Игра: кто сделает последний разлом шоколадки $m \times n$.
*   **Логика (Теория игр):**
    1.  Чтобы разломать шоколадку размером $m \times n$ на единичные дольки ($1 \times 1$), всегда нужно сделать ровно **$m \cdot n - 1$** разломов.
    2.  Это число не зависит от стратегии игроков.
    3.  Если число разломов **нечетное**, выигрывает Первый (делает 1-й, 3-й... и последний ход).
    4.  Если число разломов **четное**, выигрывает Второй.
    5.  **В коде:** Мы проверяем четность площади $m \cdot n$.
        *   Если $m \cdot n$ — четное, то $(m \cdot n - 1)$ — нечетное $\to$ победит **Первый**.
        *   Иначе $\to$ победит **Второй**.
*   **Сложность по времени:** $O(1)$.
*   **Сложность по памяти:** $O(1)$.
```
#include <iostream>

int main() {
    int m, n;
    std::cin >> m >> n;
    
    if ((m * n) % 2 == 0) {
        std::cout << "[:=[first]" << std::endl;
    } else {
        std::cout << "[second]=:]" << std::endl;
    }

    return 0;
}
```
