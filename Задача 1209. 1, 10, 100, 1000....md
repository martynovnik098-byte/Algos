**Задача 1209. 1, 10, 100, 1000...**

*   **Суть:** Определить, стоит ли на позиции $K$ цифра 1.
*   **Логика:**
    1.  Последовательность состоит из степеней десятки: $1, 10, 100, 1000 \dots$
    2.  Единицы стоят только в начале каждого нового числа. Длины чисел образуют ряд: $1, 2, 3, 4 \dots$
    3.  Позиции единиц — это сумма арифметической прогрессии $+ 1$:
        $Pos = 1 + \frac{n(n+1)}{2}$.
    4.  Задача сводится к проверке: является ли число $K-1$ "треугольным числом".
    5.  **Математика:** Решаем квадратное уравнение. Единица будет, если выражение **$D = 8 \cdot (K-1) + 1$** является полным квадратом **нечетного** числа.
*   **Алгоритм:**
    1.  Считаем `D = 1 + 8 * (K - 1)`.
    2.  Извлекаем корень: `S = sqrt(D)`.
    3.  Проверяем: если `S * S == D` (корень целый) **и** `S` нечетное — выводим 1, иначе 0.
*   **Сложность по времени:** $O(N)$ — обрабатываем $N$ запросов, внутри каждого простая математика $O(1)$.
*   **Сложность по памяти:** $O(N)$ — в данном коде используется вектор для хранения всех $K$, но можно было бы решать и "на лету" за $O(1)$.
*   **Тип данных:** Обязательно `long long`, так как $K$ до $2^{31}-1$, а при вычислении $8 \cdot K$ будет переполнение обычного `int`.
```
#include <iostream>
#include <vector>
#include <cmath>

void solve() {
    int n;
    if (!(std::cin >> n)) return;

    std::vector<long long> k_values(n);
    for (int i = 0; i < n; ++i) {
        if (!(std::cin >> k_values[i])) return;
    }

    bool first = true;
    for (long long k : k_values) {
        
        long long R = k - 1;
        long long D = 1 + 8 * R;
        
        long long S = round(sqrt((long double)D));
        
        int result = 0;
        
        if (S * S == D && S % 2 == 1) {
            result = 1;
        }

        if (!first) {
            std::cout << " ";
        }
        std::cout << result;
        first = false;
    }
    std::cout << "\n";
}

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);
    solve();
    return 0;
}
```
