**Задача 1877. Велосипедные коды**

*   **Суть:** Вор подбирает коды по порядку: `0000`, `0001`, `0002`... Ден меняет замки каждый день.
    *   В нечетные дни (1-й, 3-й...) висит **Замок 1**, вор пробует **четные** числа (0, 2...).
    *   В четные дни (2-й, 4-й...) висит **Замок 2**, вор пробует **нечетные** числа (1, 3...).
*   **Логика:** Взлом удастся, если:
    *   Код первого замка (`a`) — **четный** (`a % 2 == 0`).
    *   **ИЛИ** (`||`)
    *   Код второго замка (`b`) — **нечетный** (`b % 2 != 0`).
*   **Сложность по времени:** $O(1)$.
*   **Сложность по памяти:** $O(1)$.
```
#include <iostream>


int main() {
    int a, b;
    std::cin >> a >> b;
    if (a % 2 == 0 || b % 2 != 0) {
        std::cout << "yes" << std::endl;
    }
    else {
        std::cout << "no" << std::endl;
    }
    return 0;
}
```
