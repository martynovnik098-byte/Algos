**Задача 1820. Уральские бифштексы**

*   **Суть:** Пожарить $n$ стейков с **двух** сторон на сковороде вместимостью $k$.
*   **Логика:**
    1.  **Случай 1 ($n \le k$):** Все стейки влезают на сковороду одновременно. Жарим одну сторону (1 мин), переворачиваем, жарим вторую (1 мин). Итого: **2 минуты**.
    2.  **Случай 2 ($n > k$):** Стейков больше, чем места.
        *   Всего нужно пожарить **$2 \cdot n$** сторон.
        *   За 1 минуту мы обрабатываем **$k$** сторон.
        *   Нам нужно разделить $(2 \cdot n)$ на $k$ с **округлением вверх** (потому что нельзя пожарить часть стороны, минута тратится полностью).
*   **Формула:** `(2 * n + k - 1) / k`.
    *   Это формула целочисленного деления с округлением вверх ($\lceil \frac{A}{B} \rceil = (A + B - 1) / B$).
*   **Сложность по времени:** $O(1)$ — одна формула.
*   **Сложность по памяти:** $O(1)$.
```
#include <iostream>

int main() {
    int n, k;
    
    
    if (std::cin >> n >> k) {
        if (n <= k) {
            std::cout << 2 << std::endl;
        } else {
            int result = (2 * n + k - 1) / k;
            std::cout << result << std::endl;
        }
    }
    
    return 0;
}
```
