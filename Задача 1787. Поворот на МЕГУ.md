**Задача 1787. Поворот на МЕГУ**

*   **Суть:** Симуляция очереди (пробки) на светофоре.
*   **Логика:**
    1.  Изначально в пробке 0 машин.
    2.  Запускаем цикл на $n$ минут.
    3.  Каждую минуту добавляем приехавшие машины (`queue += ai`).
    4.  Пытаемся вычесть $k$ машин (пропускная способность).
    5.  **Условие:** Если машин больше $k$, то вычитаем $k$. Если меньше или равно $k$ — пробка исчезает (становится 0). Очередь не может быть отрицательной.
*   **Алгоритм:** Итеративный (линейный проход).
*   **Сложность по времени:** $O(n)$ — цикл выполняется $n$ раз.
*   **Сложность по памяти:** $O(1)$ — храним только текущий размер очереди, массив не нужен (считываем числа по одному).
```
#include <iostream>

int main() {
    int k, n;  

    std::cin >> k >> n;

    int queue = 0;

    for (int i = 0; i < n; i++) {
        int ai;  
        std::cin >> ai;
        queue += ai;

        if (queue > k) {
            queue -= k;  
        }
        else {
            queue = 0;  
        }
    }
    std::cout << queue << std::endl;
    return 0;
}
```
