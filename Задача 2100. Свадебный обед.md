**Задача 2100. Свадебный обед**

*   **Суть:** Посчитать стоимость банкета ($100 с человека), учитывая "плюс один" и суеверие про число 13.
*   **Логика:**
    1.  Начальное количество = **2** (сами Маршалл и Лили).
    2.  Читаем список гостей.
    3.  Если в строке есть подстрока `"+one"` (`s.find(...)`), добавляем **2** человек. Иначе — **1**.
    4.  **Проверка на 13:** Если итоговая сумма равна **13**, добавляем +1 (сажаем манекен).
    5.  Ответ: `кол-во гостей * 100`.
*   **Алгоритм:** Линейный проход с поиском подстроки.
*   **Сложность по времени:** $O(N \cdot L)$, где $N$ — число гостей, $L$ — длина имени (но так как длина ограничена 20 символами, можно считать $O(N)$).
*   **Сложность по памяти:** $O(1)$ (храним только текущую строку и счетчик).
```
#include <iostream>
#include <string>

int main() {
    int n;
    std::cin >> n;
    
    int count = 2;
    std::string s;
    
    for (int i = 0; i < n; ++i) {
        std::cin >> s;
        if (s.find("+one") != std::string::npos) {
            count += 2;
        } else {
            count += 1;
        }
    }
    
    if (count == 13) {
        count++;
    }
    
    std::cout << count * 100 << "\n";
    
    return 0;
}
```
