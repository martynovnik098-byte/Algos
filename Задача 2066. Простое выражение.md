**Задача 2066. Простое выражение**

*   **Суть:** Расставить знаки между $a, b, c$ (где $a \le b \le c$), чтобы получить **минимальный** результат.
*   **Логика:**
    1.  Чтобы результат был минимальным, нужно из самого маленького числа ($a$) вычесть как можно большее значение, составленное из оставшихся ($b$ и $c$).
    2.  Максимум из $b$ и $c$ можно получить либо сложением ($b+c$), либо умножением ($b \cdot c$).
    3.  Код проверяет два варианта:
        *   `a - b * c` (вычитаем произведение).
        *   `a - b - c` (вычитаем сумму).
    4.  Функция `std::min` выбирает наименьшее из них.
    *   *Примечание:* Другие варианты перестановок не дадут результата меньше, так как $a$ — самое маленькое число.
*   **Сложность по времени:** $O(1)$.
*   **Сложность по памяти:** $O(1)$.
```
#include <iostream>
#include <algorithm>

int main() {
    int a, b, c;
    std::cin >> a >> b >> c;
    int res = std::min(a - b * c, a - b - c);
    std::cout << res << std::endl;
    return 0;
}
```
