**Задача 1197. Один в поле воин**

*   **Суть:** Посчитать количество клеток под боем коня.
*   **Логика:**
    1.  **Координаты:** Переводим букву и цифру (например, "d4") в числовые координаты $X$ и $Y$ (от 1 до 8).
        *   `s[0] - 'a' + 1` превращает 'a' $\to$ 1, 'b' $\to$ 2 и т.д.
    2.  **Ходы коня:** У коня максимум 8 вариантов хода (буквой «Г»). Смещения по осям записаны в массивы `dx` и `dy` (комбинации $\pm 1$ и $\pm 2$).
    3.  **Перебор:** В цикле применяем все 8 смещений к текущей позиции.
    4.  **Проверка:** Если новая координата попадает в границы доски (от 1 до 8 включительно), увеличиваем счетчик.
*   **Сложность по времени:** $O(1)$ для одного теста (всегда ровно 8 итераций). Для $N$ тестов — $O(N)$.
*   **Сложность по памяти:** $O(1)$.
```
#include <iostream>
#include <string>

int main() {
    int n;
    std::cin >> n;
    while (n--) {
        std::string s;
        std::cin >> s;
        int x = s[0] - 'a' + 1;
        int y = s[1] - '0';
        int count = 0;
        int dx[] = { -2, -2, -1, -1, 1, 1, 2, 2 };
        int dy[] = { -1, 1, -2, 2, -2, 2, -1, 1 };
        
        for (int i = 0; i < 8; ++i) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            if (nx >= 1 && nx <= 8 && ny >= 1 && ny <= 8) {
                count++;
            }
        }
        std::cout << count << std::endl;
    }
    return 0;
}
```
