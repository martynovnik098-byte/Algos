**Задача 1264. Трудовые будни**

*   **Суть:** Посчитать, сколько строк кода напишет программист (1 строка = 1 секунда).
*   **Логика:**
    1.  Есть массив из $N$ элементов.
    2.  Нужно проверить каждое число от $0$ до $M$.
    3.  Количество чисел от 0 до $M$ равно **$M + 1$** (например, 0 и 1 — это 2 числа).
    4.  Для каждого из $N$ элементов массива пишется блок из $M+1$ проверок (`if`).
    5.  **Формула:** `N * (M + 1)`.
*   **Сложность по времени:** $O(1)$.
*   **Сложность по памяти:** $O(1)$.
*   **Тип данных:** `int` подходит. Максимальное значение $40000 \times 40001 \approx 1.6$ млрд, что влезает в стандартный `int` (до 2 млрд).
```
#include <iostream>

int main() {
    int N, M;
    std::cin >> N >> M;
    int total_lines = N * (M + 1);
    std::cout << total_lines << std::endl;

    return 0;
}
```
