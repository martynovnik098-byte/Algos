Вот краткая выжимка для шпоры:

**Задача 1068. Сумма**

*   **Суть:** Найти сумму всех целых чисел от $1$ до $N$.
*   **Подвох:** $N$ может быть **отрицательным**.
    *   Если $N = 5$, сумма: $1 + 2 + 3 + 4 + 5$.
    *   Если $N = -3$, сумма: $(-3) + (-2) + (-1) + 0 + 1$.
*   **Логика (в коде):**
    1.  Если $N > 0$: Запускаем цикл от 1 до $N$, прибавляя числа к сумме.
    2.  Если $N \le 0$: Запускаем цикл от $N$ до 1 (идем "вверх" к единице), прибавляя числа.
*   **Алгоритм:** Линейный проход циклом.
*   **Сложность по времени:** $O(|N|)$ — количество итераций зависит от модуля $N$. (Можно решить за $O(1)$ формулой арифметической прогрессии, но в коде — цикл).
*   **Сложность по памяти:** $O(1)$.

```
#include <iostream>

int main() {
    int N;
    std::cin >> N;

    int sum = 0;

    if (N > 0) {
        for (int i = 1; i <= N; ++i) {
            sum += i;
        }
    } else {
        for (int i = N; i <= 1; ++i) {
            sum += i;
        }
    }

    std::cout << sum << std::endl;

    return 0;
}
```
