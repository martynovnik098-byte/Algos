**Задача 1313. К вопросу о спорте**

*   **Суть:** Вывести элементы матрицы $N \times N$ по диагоналям (начиная с верхнего левого угла).
*   **Логика:**
    1.  Главное свойство диагоналей: сумма индексов координат $(i + j)$ для элементов одной диагонали — **константа**.
    2.  Перебираем сумму $k$ от $0$ до $2N-2$ (максимальная сумма индексов для $N \times N$).
    3.  Внутри каждой диагонали нам нужно идти "снизу-вверх".
        *   Начинаем с максимально возможной строки $i$ (но не больше $N-1$).
        *   Уменьшаем $i$, а столбец вычисляем как $j = k - i$.
        *   Если $j$ попадает в границы матрицы — выводим число.
*   **Алгоритм:** Проход по матрице с использованием свойства суммы индексов.
*   **Сложность по времени:** $O(N^2)$ — мы посещаем каждый из $N^2$ пикселей ровно один раз.
*   **Сложность по памяти:** $O(N^2)$ — для хранения матрицы.
```
#include <iostream>
#include <vector>
#include <cmath>

void solve() {
    int n;
    if (!(std::cin >> n)) return;

    std::vector<std::vector<int>> matrix(n, std::vector<int>(n));
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            if (!(std::cin >> matrix[i][j])) return;
        }
    }

    std::vector<int> result_sequence;
    for (int d = 0; d < 2 * n - 1; ++d) {
        int i_start = std::min(d, n - 1);
        for (int i = i_start; i >= 0; --i) {
            int j = d - i;
            if (j >= 0 && j < n) {
                result_sequence.push_back(matrix[i][j]);
            }
        }
    }
    bool first = true;
    for (int val : result_sequence) {
        if (!first) {
            std::cout << " ";
        }
        std::cout << val;
        first = false;
    }
    std::cout << "\n";
}

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);
    solve();
    return 0;
}
```
