**Задача 2012. Про Гришу Н.**

*   **Суть:** Успеет ли Гриша решить 12 задач за 5 часов (300 минут).
*   **Логика:**
    1.  Первый час (60 мин) тратится на решение $f$ задач.
    2.  Остается задач: $12 - f$.
    3.  Время на оставшиеся задачи: $(12 - f) \times 45$ минут.
    4.  **Полное время:** `60 + (12 - f) * 45`.
    5.  Если это время $\le 300$ (5 часов), выводим "YES".
*   **Сложность по времени:** $O(1)$ — простая арифметическая формула.
*   **Сложность по памяти:** $O(1)$.
*   **Лайфхак:** Математически это условие сводится к: если $f \ge 7$, то "YES", иначе "NO".
```
#include <iostream>

int main() {
    int f;  
    std::cin >> f;
    int total_time = 60 + (12 - f) * 45;
    if (total_time <= 300) {
        std::cout << "YES" << std::endl;
    }
    else {
        std::cout << "NO" << std::endl;
    }

    return 0;
}
```
