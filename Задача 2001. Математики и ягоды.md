**Задача 2001. Математики и ягоды**

*   **Суть:** Вычислить чистый вес ягод, вычитая вес пустых корзин.
*   **Логика:**
    1.  Входные данные дают 3 взвешивания.
    2.  Во 2-м взвешивании (`a2 b2`) вторая корзина пуста, значит **$b_2$ — это вес пустой корзины 2**.
    3.  В 3-м взвешивании (`a3 b3`) первая корзина пуста, значит **$a_3$ — это вес пустой корзины 1**.
    4.  $a_1$ и $b_1$ — это полные веса (ягоды + корзина).
*   **Формула:**
    *   Ягоды 1: `a1 - a3` (общий вес минус пустая тара).
    *   Ягоды 2: `b1 - b2`.
*   **Сложность:** $O(1)$ по времени и памяти.
```
#include <iostream>

int main() {
    int a1, b1;  
    int a2, b2;  
    int a3, b3;  

    std::cin >> a1 >> b1;
    std::cin >> a2 >> b2;
    std::cin >> a3 >> b3;

    int B1 = a3;  
    int B2 = b2;  

    int M1 = a1 - B1;  
    int M2 = b1 - B2;  

    std::cout << M1 << " " << M2 << std::endl;

    return 0;
}
```
