**Задача 1880. Собственные числа Psych Up**

*   **Суть:** Найти количество общих чисел, которые встречаются **во всех трех** массивах одновременно.
*   **Логика:**
    1.  Массивы уже **отсортированы** (это важно!).
    2.  Используем метод **трех указателей** (итераторов `it1`, `it2`, `it3`).
    3.  В цикле смотрим на текущие элементы всех трех массивов:
        *   Если они **равны**: Ура, нашли общее число! Увеличиваем счетчик и двигаем **все три** указателя вперед.
        *   Если **не равны**: Находим среди них **наименьшее** и двигаем только его указатель вперед (потому что в отсортированном массиве это число уже точно не совпадет с большими числами в других массивах).
*   **Алгоритм:** Поиск пересечения отсортированных множеств (Two/Three Pointers).
*   **Сложность по времени:** $O(N)$ — линейная. Мы проходим по каждому массиву один раз (где $N$ — суммарное количество элементов во всех массивах).
*   **Сложность по памяти:** $O(N)$ — нужно хранить сами массивы в памяти.

```
//1880. Собственные числа Psych Up
#include <iostream>
#include <vector>

std::vector<int> read_numbers() {
    int n;
    std::cin >> n;
    std::vector<int> numbers(n);
    for (int i = 0; i < n; ++i) {
        std::cin >> numbers[i];
    }
    return numbers;
}

int main() {
    std::vector<int> nums1 = read_numbers();
    std::vector<int> nums2 = read_numbers();
    std::vector<int> nums3 = read_numbers();
    auto it1 = nums1.begin();
    auto it2 = nums2.begin();
    auto it3 = nums3.begin();
    int counter = 0;
    while (it1 != nums1.end() && it2 != nums2.end() && it3 != nums3.end()) {
        if (*it1 == *it2 && *it2 == *it3){
            ++counter;
            ++it1;
            ++it2;
            ++it3;
        }
        else{
            if (*it1 <= *it2 && *it1 <= *it3){
                ++it1;
            }
            else if (*it2 <= *it1 && *it2 <= *it3){
                ++it2;
            }
            else if (*it3 <= *it1 && *it3 <= *it2){
                ++it3;
            }
        }
    }
    std::cout << counter;
    return 0;
}
```
